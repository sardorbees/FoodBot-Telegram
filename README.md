🚀 Быстрый старт
Вариант 1 — через Docker (рекомендуется)
bash# 1. Скопируйте проект
git clone https://github.com/yourname/foodbot.git
cd foodbot

# 2. Настройте переменные окружения
cp .env.example .env
nano .env   # заполните BOT_TOKEN и ADMIN_IDS

# 3. Запустите контейнеры
docker-compose up -d

# 4. Заполните БД начальными данными
docker-compose exec bot python seed.py
Вариант 2 — локально
bash# 1. Установите зависимости
pip install -r requirements.txt

# 2. Настройте .env
cp .env.example .env

# 3. Убедитесь что PostgreSQL запущен, затем:
python seed.py   # начальные данные
python main.py   # запуск бота

⚙️ Переменные окружения
Создайте файл .env на основе .env.example:
env# Telegram
BOT_TOKEN=123456:ABC-DEF...        # Токен от @BotFather
ADMIN_IDS=123456789,987654321      # Telegram ID администраторов (через запятую)

# PostgreSQL
DB_HOST=localhost
DB_PORT=5432
DB_NAME=foodbot
DB_USER=postgres
DB_PASS=yourpassword

# Доставка
DELIVERY_COST=10000                # Стоимость доставки в сумах
FREE_DELIVERY_FROM=100000          # Сумма заказа для бесплатной доставки

# Поддержка
SUPPORT_USERNAME=@your_support

# Платёжные системы (заполните при подключении)
CLICK_MERCHANT_ID=
CLICK_SERVICE_ID=
PAYME_MERCHANT_ID=
PAYME_SECRET_KEY=
UZUM_MERCHANT_ID=

Как получить BOT_TOKEN: откройте @BotFather → /newbot → следуйте инструкциям.


Как узнать свой Telegram ID: напишите @userinfobot.

📦 Модули системы
Поток заказа клиента
/start → Главное меню
  └─ 🍔 Меню
       └─ Категория (Пицца / Бургеры / Суши...)
            └─ Карточка блюда (фото, цена, вес, время)
                 └─ Выбор размера / Добавление допов
                      └─ 🛒 Корзина
                           ├─ Промокод
                           └─ ✅ Оформить заказ
                                ├─ 1. Адрес / геолокация
                                ├─ 2. Номер телефона
                                ├─ 3. Время доставки
                                ├─ 4. Способ оплаты
                                └─ 5. Комментарий → Заказ создан!
Статусы заказа
СтатусОписание🆕 newНовый заказ, ждёт подтверждения👨‍🍳 preparingГотовится на кухне📦 courier_assignedПередан курьеру🚴 in_deliveryКурьер в пути✅ deliveredДоставлен❌ cancelledОтменён
При каждом изменении статуса клиент получает автоматическое уведомление.
Система лояльности

Бонусные баллы — начисляются за каждый выполненный заказ
Кэшбек — 3% от суммы заказа возвращается на баланс
Промокоды — скидка в % или фиксированная сумма, с лимитом использования
Реферальная система — приведи друга и получи 200 баллов, друг получит 100
VIP-уровни — Стандарт → Серебро → Золото → VIP 👑

🖥 Деплой на VPS

# 2. Скопируйте проект на сервер

# 4. Запуск

❓ FAQ
Как добавить нового курьера?
sqlUPDATE users SET role = 'courier' WHERE telegram_id = 123456789;
Как добавить блюдо?
Войдите в бот как администратор → нажмите «🍽 Меню» → «➕ Добавить блюдо» и следуйте шагам.
Бот не отвечает — что делать?
bashdocker-compose logs bot   # посмотрите ошибки
docker-compose restart bot
Как сделать резервную копию БД?
bashdocker-compose exec db pg_dump -U postgres foodbot > backup.sql
